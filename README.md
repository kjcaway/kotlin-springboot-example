## Spring boot example with kotlin
Backend code example using spring boot, jpa, swagger, async, junit test.

---

### Environment & Lib
- Spring Boot 2.4.1
- Spring Web MVC
- Spring Data Jpa
- Swagger
- RDB(MariaDB)

### Description
- Boilerplate for Spring Web MVC
- Example Code
    - AOP
    - Filter, Interceptor
    - JPA, Fetch Join
    - RestTemplate
    - Async, Future
    - Exception Handle
    - Unit Test, Integration Test
